ABSTRACT
------------------------------------------------------------
The Hand Gestures Fun Projects is a collection of interactive,
web-based games that use hand gestures as the primary mode of
control. These projects explore natural human-computer
interaction by replacing traditional input devices such as
keyboards and mice with real-time hand gesture recognition
using a webcam. The system enhances user engagement and
demonstrates the application of computer vision concepts in
browser-based environments.

------------------------------------------------------------
1. INTRODUCTION
------------------------------------------------------------
Human-Computer Interaction (HCI) has evolved rapidly with the
integration of computer vision techniques. Hand gesture
recognition allows users to interact with digital systems in
a natural and intuitive manner. This project focuses on
implementing fun and educational games that respond to hand
movements and gestures, making interaction more immersive and
accessible.

------------------------------------------------------------
2. OBJECTIVES
------------------------------------------------------------
• To design interactive games controlled using hand gestures
• To explore real-time hand tracking using a webcam
• To improve understanding of browser-based computer vision
• To enhance user engagement through gesture-based control
• To demonstrate practical HCI applications

------------------------------------------------------------
3. SYSTEM OVERVIEW
------------------------------------------------------------
The system consists of multiple HTML-based applications, each
representing a unique game. The webcam captures real-time hand
movements, which are processed using hand-tracking techniques.
Detected gestures are mapped to game actions, enabling users
to interact without physical controllers.

------------------------------------------------------------
4. PROJECT MODULES
------------------------------------------------------------

4.1 Advanced Canvas Game (Advcanvas.html)
------------------------------------------------------------
This module uses an HTML5 canvas to create an interactive
environment where users can control or manipulate elements
using hand movements. It demonstrates advanced canvas usage
combined with gesture-based input.

4.2 Air Game (air.html)
------------------------------------------------------------
The Air Game allows users to control in-game objects using
hand motion. The movement of the hand in air is mapped to
directional or positional controls, creating a touch-free
gaming experience.

4.3 Rock Paper Scissors (rockpaper.html)
------------------------------------------------------------
This module implements the classic Rock-Paper-Scissors game
using hand gestures. The system detects specific hand shapes
and determines the winner based on predefined rules.

4.4 Try Piano (trypiano.html)
------------------------------------------------------------
The Try Piano module enables users to play virtual piano keys
using hand gestures. Different hand positions trigger musical
notes, offering an interactive and creative musical
experience.
